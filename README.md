# react-ts-sample-app

## 環境

- React 17.0.2
- TypeScript 4.4.2
- Emotion 11.7.1
- axios 0.24.0

## 追加の前提条件

- ユーザー一覧を取得する API は、Backlog API のユーザー一覧取得 API を想定
  - https://developer.nulab.com/ja/docs/backlog/api/2/get-user-list
- IE はサポート外とする
